# Image Gallery Lightbox

A simple image gallery where clicking a thumbnail opens a larger image in a modal.

---

## Features

* Responsive grid layout
* Click image to view in lightbox
* Close button and outside click support
* Smooth transition effects

---

## Technologies

* HTML
* CSS
* JavaScript

---

## Project Structure

```
project-folder/
│── index.html
│── styles.css
│── script.js
```

---

## How It Works

* Thumbnails are displayed using CSS Grid
* Clicking an image updates the lightbox image source
* A CSS class (`active`) controls visibility

---

## Concepts Covered

* DOM selection (`querySelector`, `querySelectorAll`)
* Event listeners
* CSS Grid and transitions
* Class manipulation in JavaScript

# Dynamic Quiz Application

A simple quiz web app built using HTML, CSS, and JavaScript. It dynamically loads questions, captures user answers, and displays the final score.

---

## Features

* Dynamic question loading from JavaScript
* Interactive option selection with UI feedback
* Highlights correct and wrong answers
* Score tracking and result display

---

## How It Works

1. Questions are stored in a JavaScript array.
2. Each question is displayed one by one.
3. User selects an option.
4. The app checks:

   * Correct → shows green
   * Wrong → shows red and highlights correct answer
5. Moves to next question.
6. Final score is displayed at the end.
